Данная программа представляет собой консольное приложение для управления личными финансами. Она позволяет пользователям вести учет своих доходов и расходов, сохраняя записи в текстовом файле.

Основные функции:

    1. Вывод баланса (print_balance): Подсчитывает и выводит общий баланс пользователя, а также отдельно суммы доходов и расходов, основываясь на данных из списка записей.
    2. Добавление записи (add_record): Запрашивает у пользователя данные о новой финансовой операции, включая дату, категорию (доход или расход), сумму и описание. Проверяет введенные данные на соответствие формату и добавляет запись в список.
    3. Редактирование записи (edit_record): Позволяет пользователю изменить существующую запись. Пользователь выбирает запись по номеру, затем вводит новые данные для обновления.
    4. Поиск по записям (search_records): Предоставляет возможность поиска записей по дате, категории или сумме. Выводит все записи, соответствующие заданным критериям.
    5. Сохранение записей (save_records): Сохраняет текущий список записей в текстовый файл records.txt, перезаписывая его содержимое.
    6. Загрузка записей (load_records): Загружает записи из текстового файла records.txt в список при старте программы.

Механика программы:

    • При запуске программа сначала пытается загрузить существующие записи из файла records.txt с помощью функции load_records.
    • Пользователю предлагается меню с выбором действий: вывод баланса, добавление, редактирование и поиск записей, а также выход из программы.
    • В зависимости от выбора пользователя программа вызывает соответствующую функцию для выполнения действия.
    • После выполнения действия (кроме выхода) программа возвращается в главное меню, позволяя пользователю совершить другие операции.
    • Все изменения, внесенные в список записей (добавление или редактирование), автоматически сохраняются в файле records.txt.
    • При выходе из программы текущее состояние списка записей сохраняется, что позволяет вести непрерывный учет финансов.

Программа использует простую текстовую базу данных (файл records.txt), что делает ее легкой для использования и не требует сложной настройки баз данных. Она также включает в себя валидацию ввода пользователя, чтобы обеспечить корректность данных.
